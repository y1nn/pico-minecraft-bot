## How to Contribute

### 1. Fork the Repository
Click the "Fork" button at the top right of the repository page.

### 2. Clone Your Fork
```bash
git clone https://github.com/YOUR_USERNAME/pico-minecraft-bot.git
cd pico-minecraft-bot
```

### 3. Create a Branch
```bash
git checkout -b feature/your-feature-name
```

### 4. Make Your Changes
- Follow the existing code style.
- Add comments for complex logic.
- Test your changes before committing.

### 5. Commit Your Changes
```bash
git add .
git commit -m "Add: Your feature description"
```

### 6. Push and Create a Pull Request
```bash
git push origin feature/your-feature-name
```
Then open a Pull Request on GitHub.

---

## Guidelines

- **Keep it simple**: This is a lightweight bot. Avoid heavy dependencies.
- **Security first**: Never hardcode secrets. Use environment variables.
- **Test locally**: Make sure the bot runs without errors before submitting.
